### Aufgabe 1: JSON laden und anzeigen
Bearbeitungszeit: ca. 25 Minuten


### Ausgangslage
Es gibt:
• eine HTML-Datei
• eine JSON-Datei auf dem Server (z. B. data.json)
• ein leeres <div> für die Ausgabe
Die JSON-Datei enthält mehrere Objekte (z. B. Bücher, Produkte, Personen).


#### Aufgabe
1. Lade die JSON-Datei per AJAX.
2. Parse die Antwort in ein JavaScript-Objekt.
3. Gib alle Einträge strukturiert im HTML aus:
a. mindestens ein Wert pro Objekt
b. übersichtlich (z. B. Liste oder Absätze)

#### Hinweise:
• Nutze JSON.parse()
• Keine Tabellen nötig
